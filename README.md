# wordcraft
Wordcraft - Gerador Profissional de Caça-Palavras é uma ferramenta web moderna e intuitiva projetada para educadores, pais e entusiastas de puzzles. 
Diferente de geradores antigos e complexos, o Wordcraft foca na simplicidade, permitindo criar, personalizar e exportar caça-palavras de alta qualidade em segundos.

---

## Principais Funcionalidades

- **Geração Instantânea**: Algoritmo otimizado para encaixe inteligente de palavras.
- **Personalização Total**: Defina títulos, níveis de dificuldade e listas de palavras personalizadas.
- **Inteligência de Normalização**: Tratamento automático de acentos e caracteres especiais.
- **Modo Gabarito**: Visualize a solução do puzzle antes de exportar.
- **Exportação Premium**: Baixe seus desafios em formatos PNG ou PDF prontos para impressão.
- **Design Responsivo**: Experiência otimizada para Desktop, Tablets e Smartphones.

---

## Tecnologias Utilizadas

- **Framework**: Vue 3 in Vite
- **Estilização**: Tailwind CSS (com foco em semântica e acessibilidade)
- **Gerenciamento de Estado**: Pinia
- **Componentes**: Headless UI & Heroicons
- **Exportação**: html2canvas & jsPDF

---

## Como Executar Localmente

### 1. Instalação de Dependências

```bash
npm install
```

### 2. Servidor de Desenvolvimento

Inicie o projeto em http://localhost:3000:

```bash
npm run dev
```
